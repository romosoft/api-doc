### 常量定义

### 消息类型
```js
const types = {
  TYPE_TEXT: "文本",
  TYPE_IMAGE: "图片",
  TYPE_VIDEO: "视频",
  TYPE_VOICE: "语音",
  TYPE_FILE: "文件",
  TYPE_TEMPLATE_CARD: "模板卡片",
  TYPE_DIY_CARD: "自定义卡片",
  TYPE_DIY_MSG: "自定义消息"
}
```

### 枚举常量定义
|  类型   | 枚举值  |
|  ----  | ----  |
| 消息类型  | TYPE_TEXT -- 文本 <br> TYPE_IMAGE -- 图片 <br> TYPE_VIDEO -- 视频 <br> TYPE_VOICE -- 语音 <br> TYPE_FILE -- 文件 <br> TYPE_TEMPLATE_CARD -- 模板卡片 <br> TYPE_DIY_CARD -- 自定义卡片 <br> TYPE_DIY_MSG -- 自定义消息 |
